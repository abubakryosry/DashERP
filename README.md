# DashERP

## Overview
DashERP is a mini ERP system built with React.js for managing employee records. It provides a user-friendly interface to view, add, and manage employee data effectively.

## Features
- Display all employees in a responsive table format.
- A form to add new employees.
- Detailed view of each employee's information.
- Clean and modern UI/UX design.
